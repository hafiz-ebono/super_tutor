    In `regenerate_section()` (currently lines 275-323), replace the notes lookup block with source_content lookup.

    **Current code (lines 288-295):**
    ```python
    state = (existing.session_data or {}).get("session_state", {})
    notes = state.get("notes", "")
    if not notes:

            status_code=404,
            detail=f"Session '{session_id}' has no notes. Cannot regenerate.",
    ```

    **Replace with:**
    ```python

            status_code=404,
            detail=f"Session '{session_id}' has no source content. Cannot regenerate.",
        )

    input_text = source_content
    ```

    - `notes` → `source_content` variable name
